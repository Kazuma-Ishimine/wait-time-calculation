- 初期取得で画面全体をローディング
- タイトル、平均待ち時間、入退場を表示]
- リストで表示
- リストをクリックで、詳細画面に遷移

# 詳細画面

- 初期取得で画面全体をローディング
- タイトルのテキスト入力を表示
- 説明のテキスト入力を表示
- タイトルのテキスト入力の初期値は、該当の値
- 説明のテキスト入力の初期値も該当の値
- タイトルに必須マーク
- 平均待ち時間を表示
- 保存ボタンを表示
- 削除ボタンを表示
- 保存を押すと
  - 成功時に、更新
  - 成功時に、詳細画面で変更後の値になっている
  - 保存時はボタンのみローディング
  - 画面全体はロックしない
- 初期状態は「入場」ボタン
- 入場押下後、「退場」ボタンに切り替え
- 退場押下後、「入場」ボタンに戻る
- API通信中はボタンdisabled
- 削除ボタンを押すと
  - 削除ダイアログを表示
- 退場ボタンを押すと、
  - 成功時、平均待ち時間を更新する
- 平均待ち時間は退場時に再計算
- 詳細画面表示時にも最新値を取得

# 作成画面

- タイトルのテキスト入力を表示
- 説明のテキスト入力を表示
- タイトルのテキスト入力の初期値は、空
- 説明のテキスト入力の初期値は、空
- タイトルに必須マーク
- 作成ボタンが表示
- 作成ボタン
  - 押して、作成に成功すると、詳細画面に遷移
  - ローディングが表示され、作成ボタンを非表示
  - トーストで、作成しましたと表示

# 削除ダイアログ

- タイトルを本当に削除しますか？と表示
- 警告文を表示(復元できません。)
- 削除ボタンを表示
- キャンセルボタンを表示
- 外側クリックで閉じる
- 削除ボタンを押すと、
  - 成功時に、削除
  - 成功時に、一覧画面に遷移
  - トーストで、削除しましたと表示

# バリデーション

- onChange、onSubmit時にエラー表示
- タイトル
  - 空の時、エラー
  - 101文字以上の時、エラー
- 説明
  - 401文字以上の時、エラー
